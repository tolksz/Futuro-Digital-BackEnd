# Futuro Digital — Backend Python & Django Training

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Django](https://img.shields.io/badge/Django-Framework-green)
![REST API](https://img.shields.io/badge/API-REST-orange)

Programa de formação em desenvolvimento **Back-end com Python e Django**, realizado através do projeto **Futuro Digital**, oferecido pelo Instituto Hardware e IFRS em parceria com o Governo Federal apoiado pela SOFTEX.

---

## 🎯 Objetivo

Este repositório tem como finalidade acompanhar a evolução técnica ao longo do curso, reunindo exercícios práticos, estudos dirigidos e projetos relacionados ao desenvolvimento backend utilizando Python e Django.

---

## 📚 Conteúdo do Curso

### Base de Programação
- Lógica de programação
- Programação estruturada em Python

### Orientação a Objetos (POO)
- Classes e objetos
- Encapsulamento
- Herança
- Organização de código orientado a objetos

### Django — Setup Inicial

Uso do framework Django, incluindo:

- Instalação do ambiente
- Configuração de ambiente virtual
- Criação e estruturação de projetos Django

### Django Models

- Modelagem de banco de dados
- Migrations
- ORM (Object Relational Mapping)

### APIs REST

Construção de APIs utilizando Django REST Framework.
